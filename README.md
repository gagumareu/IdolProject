## 개발 목표
k-pop 아이돌 통합 커뮤니티 사이트
- [메인 페이지]
- 가수 상세 정보 
- 회원 전용 커뮤니티 포럼
- 회원 랭킹 시스템(활동 횟수)
- 회원 중고거래 포럼
- [관리자 모드]
- 가수, 음원, 이벤트, 공지 등록, 수정, 삭제 모드
- 회원 정보 상제 조회
- 회원 1:1 문의 답변

## 사용기술
- 서버 : apache - tomcat - 9.0.11

- Spring Tool Suite 3

- Database : Oracle SQL Developer

- 사용 언어 : Java, HTML, CSS, Javascript, Jquery, JSP, SQL

- 버전 관리 : Git



## Advanced Feature
- 이미지 파일 다중 등록
- 다중 파일 수정 시 다중 등록 및 기존 파일 삭제
